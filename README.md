# QareebShare Android — قريب شير

تطبيق Android أصلي وخفيف لمشاركة الملفات والتطبيقات بين هاتفين قريبين دون إنترنت أو راوتر.

## حالة المشروع

وصل المشروع إلى **الدفعة الثانية**، وتشمل:

- مشروع Java وXML أصلي دون مكتبات تشغيل خارجية.
- دعم Android 6 (API 23) حتى Android 17 (API 37).
- واجهة عربية أصلية تدعم RTL والوضع الداكن.
- اختيار ملف واحد أو عدة ملفات عبر محدد ملفات أندرويد.
- عرض التطبيقات المثبتة غير التابعة للنظام واختيارها.
- طلب صلاحيات الأجهزة القريبة حسب إصدار أندرويد.
- اكتشاف أجهزة Wi‑Fi Direct وعرضها.
- وضع استقبال ينشئ مجموعة مباشرة دون راوتر.
- الاتصال بجهاز قريب وتحديد Group Owner.
- مصافحة TCP محلية للتأكد من جاهزية قناة النقل.
- إيقاف وتنظيف الاتصال والمجموعة عند الإلغاء.
- بناء GitHub Actions يدوي فقط أثناء اكتمال دفعات التطوير.

نقل الملفات الفعلي، التقدم، التحقق من الملفات، الاستكمال، التشفير، ومشاركة Split APKs ستضاف في الدفعات التالية.

## بيئة البناء

- Android Gradle Plugin: 9.2.0
- Gradle: 9.4.1
- JDK: 17 لتشغيل نظام البناء
- Java source compatibility: 8
- compileSdk / targetSdk: 37
- minSdk: 23

## البناء عبر GitHub

البناء متوقف عن العمل التلقائي أثناء تطبيق الدفعات.
عند الحاجة افتح:

`Actions → Android Build → Run workflow`

وبعد نجاحه ستجد APK داخل:

`Artifacts → qareeb-share-apks`

## الفحص دون بناء

```bash
./scripts/check-batch2.sh
```

لا يحتاج تطبيق الإنتاج إلى الإنترنت أو حساب أو خادم أو Google Play Services.
