# QareebShare Android — قريب شير

تطبيق Android أصلي وخفيف لمشاركة الملفات والتطبيقات بين هاتفين قريبين دون إنترنت أو راوتر.

## حالة المشروع

وصل المشروع إلى **الدفعة الثالثة**، وتشمل:

- Java وXML أصليان دون مكتبات تشغيل خارجية.
- دعم Android 6 (API 23) حتى Android 17 (API 37).
- واجهة عربية تدعم RTL والوضع الداكن.
- اختيار ملف واحد أو عدة ملفات عبر محدد ملفات أندرويد.
- عرض التطبيقات المثبتة غير التابعة للنظام واختيارها.
- اكتشاف أجهزة Wi-Fi Direct والاتصال بها دون راوتر.
- ServerSocket وSocket محليان على المنفذ 8988.
- طلب قبول أو رفض قبل استقبال أي محتوى.
- نقل Streaming متعدد الملفات دون تحميل الملف كاملاً في الذاكرة.
- إرسال Base APK وSplit APKs للتطبيقات المختارة.
- حفظ النتائج في مجلد QareebShare داخل Downloads.
- SHA-256 للتحقق من سلامة كل عنصر قبل اعتماده.
- تقدم فعلي وسرعة نقل وإشعار خدمة أمامية.
- إلغاء النقل وتنظيف الاتصال والملفات غير المكتملة.
- GitHub Actions يدوي فقط حتى اكتمال المشروع.

الدفعات التالية ستضيف الاستكمال بعد الانقطاع، تشفير الجلسة، تثبيت التطبيقات المستلمة، والسجل المحلي.

## بيئة البناء

- Android Gradle Plugin: 9.2.0
- Gradle: 9.4.1
- JDK: 17 لتشغيل نظام البناء
- Java source compatibility: 8
- compileSdk / targetSdk: 37
- minSdk: 23

## البناء عبر GitHub

البناء لا يعمل تلقائياً أثناء تطبيق الدفعات. بعد اكتمال المشروع يفتح المستخدم:

`Actions → Android Build → Run workflow`

وبعد نجاحه يوجد APK داخل:

`Artifacts → qareeb-share-apks`

## الفحص دون بناء

```bash
./scripts/check-batch3.sh
```

لا يحتاج تطبيق الإنتاج إلى الإنترنت أو حساب أو خادم أو Google Play Services.
