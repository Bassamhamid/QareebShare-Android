# الدفعة الأولى: الأساس الأصلي

## المنجز

1. إنشاء مشروع Android أصلي بوحدة تطبيق واحدة.
2. عدم استخدام AndroidX أو Compose أو Kotlin أو مكتبات خارجية.
3. واجهة رئيسية عربية خفيفة.
4. شاشة إرسال تسمح باختيار ملفات حقيقية من النظام.
5. شاشة تطبيقات تعرض التطبيقات غير النظامية فقط.
6. شاشة استقبال تفحص دعم Wi‑Fi Direct على الجهاز.
7. شاشة سجل محلية بحالة فارغة صحيحة.
8. إعداد صلاحيات الاتصال المخطط لها حسب إصدارات أندرويد.
9. إعداد R8 وتقليص الموارد في إصدار Release.
10. إعداد GitHub Actions لفحص وبناء APK.

## ما لم يُنفّذ بعد

- اكتشاف الأجهزة القريبة.
- إنشاء اتصال Wi‑Fi Direct.
- TCP Socket.
- إرسال واستقبال المحتوى.
- التشفير والاستكمال.
- تثبيت حزم APK المقسمة.

هذه العناصر موزعة على الدفعات التالية وليست وظائف وهمية داخل هذه الدفعة.
