# إعادة البناء — الدفعة الأولى

هذه الدفعة تستبدل تدفق الإرسال/الاستقبال القديم بتجربة جلسة اتصال واحدة:

- فحص حالة Wi‑Fi قبل البحث.
- فتح لوحة Wi‑Fi عندما يكون متوقفاً.
- اكتشاف هواتف قريب شير فقط عبر Wi‑Fi Direct DNS-SD.
- استجابة فورية عند الضغط على الهاتف.
- تنظيف الاتصالات العالقة قبل كل محاولة.
- مهلة اتصال وإلغاء ومحاولة جديدة.
- قناة جلسة TCP مستقلة مع نبضات للتحقق من بقاء الاتصال.
- شاشة جلسة نشطة حتى يضغط المستخدم «إنهاء الجلسة».
- مظهر تلقائي أو فاتح أو داكن.
- رسائل عربية مفهومة فقط.

هذه الدفعة مخصصة لإثبات الاتصال والجلسة على هاتفين. نقل الملفات داخل الجلسة يأتي في الدفعة الثانية بعد نجاح اختبار الاتصال الفعلي.
